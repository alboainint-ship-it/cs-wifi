WiFi Technology

WiFi is a wireless networking technology that uses radio waves to transmit data between devices and a router.

The router connects to an ISP (Internet Service Provider) and distributes internet access wirelessly.

WiFi signals travel through the air and can be affected by distance, walls, and interference from other devices.

Modern WiFi standards (WiFi 4, 5, 6) improve speed, stability, and the number of devices that can connect at once.

Common uses include browsing, streaming, gaming, and online communication.

Why Security Matters

WiFi networks can be targeted by attackers if they are not properly secured.

  • Hackers can intercept unencrypted data
  • Malware can spread through unsafe networks
  • Fake hotspots can trick users into connecting
  • Weak passwords make networks easier to break into

Security protocols like WPA2 and WPA3 encrypt network traffic to protect users.

Using strong passwords and avoiding unknown public WiFi helps reduce risk.

Encryption

Encryption is the process of converting readable data (plaintext) into unreadable data (ciphertext).

It ensures that even if data is intercepted, it cannot be understood without the correct key.

There are two main types:

  • Symmetric encryption: same key for encryption and decryption
  • Asymmetric encryption: uses a public and private key pair

Encryption is used in messaging apps, banking systems, emails, and secure websites (HTTPS).

It is a key part of protecting privacy in the digital world.

Caesar Cipher

The Caesar Cipher is one of the simplest encryption methods in which letters are shifted by a fixed number.

For example, with a shift of 3: HELLO → KHOOR.

It works by wrapping around the alphabet (Z goes back to A).

Although it shows the idea of encryption, it is not secure today.

Attackers can easily break it by trying all possible shifts (brute force attack).

It is mainly used for teaching basic cryptography concepts.

Public & Private Keys

Asymmetric encryption uses two different keys: a public key and a private key.

The public key is shared openly and is used to encrypt messages.

The private key is kept secret and is used to decrypt messages.

This ensures that even if someone intercepts the message, they cannot read it.

It is widely used in secure communication systems like HTTPS, digital signatures, and secure emails.
